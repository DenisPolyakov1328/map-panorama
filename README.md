# Проект карты с панорамой

Деплой проекта: https://map-panorama.vercel.app

Одностраничное приложение с использованием OpenLayers и Three.js.  
В проекте визуализируются объекты из GeoJSON, поддерживаются интерактивные события, есть панорама с возможностью управления.

## Функциональность

### Работа с объектами на карте
- Подсветка объекта при наведении и вывод координат в тултипе.
- Отображение данных объекта в боковой панели при клике.
- Кластеризация семафоров с совпадающими координатами.
- Для кластера в сайдбаре отображаются все элементы внутри него.

### Панорама
- Двойной клик по объекту открывает панораму и скрывает боковую панель.
- Закрытие панорамы возвращает боковую панель с данными выбранного объекта.
- Панорама поддерживает вращение, приближение, отдаление и сброс позиции.
- Добавлены кнопки управления панорамой с тултипами.
- Управление доступно как мышью, так и кнопками.

### Тултипы
- Светофоры отображают собственные координаты.
- Полигоны отображают усреднённые координаты.
- Линии отображают ближайшую координату из набора геометрии, соответствующую положению курсора.
